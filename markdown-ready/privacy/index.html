Markdown Ready Privacy Policy

Effective date: June 17, 2026

Overview

Markdown Ready is a macOS app for converting PDFs and document images into Markdown. The app is designed to process selected documents locally on your Mac.

Document Processing

Markdown Ready does not upload your selected PDFs, images, OCR text, Markdown output, or JSON sidecar output to HydRoMo servers. Conversion is performed on device using the app's local OCR workflow.

Information Stored on Your Mac

Markdown Ready may store local app settings so it can restore your workflow. This can include selected input and output locations through macOS security-scoped bookmarks, OCR settings, output format settings, local conversion history, and hashed file fingerprints used to avoid repeated folder processing.

These records are stored locally on your Mac by the app. File fingerprints are stored as hashes rather than raw fingerprint source strings.

Data Collection

HydRoMo does not collect personal data from Markdown Ready. The app does not use an account system, analytics tracking, advertising tracking, or cloud OCR service for document conversion.

Support Email

If you email HydRoMo for support, your email address and the information you include in your message will be used to respond to your request.

Contact

For privacy questions about Markdown Ready, contact support@hydromo.dev.
