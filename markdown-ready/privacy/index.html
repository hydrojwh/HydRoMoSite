Markdown Ready Privacy Policy

Last updated: June 18, 2026

Markdown Ready is a macOS document conversion app by HydRoMo. It converts PDFs, scanned pages, and document images into Markdown and optional JSON files using local processing on your Mac.

Data Collection

Markdown Ready does not collect data from the app. The app does not include account login, advertising, tracking, analytics, or a cloud OCR API for document conversion.

Document Processing

Documents selected in Markdown Ready are processed locally on your Mac. Document contents are not uploaded by Markdown Ready for OCR conversion. Output files are written to the output folder you choose.

Local Storage

Markdown Ready may store local settings on your Mac, including selected input/output paths, security-scoped bookmarks granted through macOS file and folder pickers, output preferences, and local processed-file fingerprints used to avoid duplicate folder conversions. These values are stored locally for app functionality.

Folder Access

Markdown Ready accesses files and folders only through permissions you grant using macOS file and folder pickers. You can change selected folders in the app.

Support

If you contact support, you may choose to provide information such as your email address, app version, macOS version, Mac model, screenshots, logs, or sample files. Support information is used to respond to your request. Do not send sensitive documents unless you intentionally choose to share them.

Third-Party Components

Markdown Ready includes open source software and local OCR model components. License notices are available in the app under About / Open Source Licenses.

Changes

This policy may be updated as Markdown Ready changes. If app data practices change, App Store privacy information and this policy should be updated accordingly.

Contact

For privacy or support questions, contact HydRoMo at support@hydromo.dev.
