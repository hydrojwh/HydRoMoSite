Supported Inputs

  • PDF documents
  • JPG and JPEG images
  • PNG images and screenshots

Common Checks

  • Use the macOS file or folder picker to choose an input file or input folder.
  • Choose a writable output folder before starting conversion.
  • For folder processing, leave the app running while new files are scanned.
  • If conversion quality is poor, try a higher quality source file or a cleaner scan.
