PDF to Markdown for Mac

Markdown Ready

Convert PDFs, scans, and screenshots into clean Markdown for AI workflows on your Mac.

Get Support Privacy Policy

Local Processing

Built for private document work

Markdown Ready uses an on-device OCR model on Apple Silicon. Files are selected through macOS file and folder pickers, processed locally, and written to an output folder you choose.

Features

Current App Capabilities

Markdown output

Convert PDF pages and document images into reviewable Markdown for GPT, Claude, Notion, Obsidian, RAG, and writing workflows.

Image input

Use PDF, JPG, JPEG, and PNG files. Folder processing can scan supported files and avoid repeated work with local file fingerprints.

Table options

Choose whether OCR should preserve tables and use PDFKit-based table recovery for text-layer PDFs where available.

JSON sidecar

Optionally write a JSON sidecar with source, settings, page Markdown, and simple block metadata alongside Markdown output.

Availability

Mac app in TestFlight build-up

Markdown Ready is being prepared as a macOS App Store product by HydRoMo. It works best on Apple Silicon Macs. Complex scans, handwriting, historical documents, and low-quality images are handled best effort depending on source quality.
